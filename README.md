# Chat App

## Specification

Backend:

* Node.js
* Express.js
* MongoDB
* Passport-JWT

Frontend:

* React.js
* Create React App
* Redux (Router, Thunk)
* Scss
* Webpack

## Running

```bash
# Install the dependencies.
$ cd .
$ npm install
$ cd ./client
$ npm install

# Run app on root director
$ npm run dev
```
